       ```python
       OVERLAP_CHARS = 2_000     # overlap between chunks for context continuity
       ```

       **PRICING dict** (verified 2026-03-28):
       ```python
       PRICING = {
           "claude-haiku-4-5": {"input": 1.0 / 1_000_000, "output": 5.0 / 1_000_000},
       ```

       **`clean_transcript(text, system_prompt, context_text, model, api_key, progress_queue, task_id, timeout=300.0) -> dict`:**
       - Creates `anthropic.Anthropic(api_key=api_key)` client.
       - If `context_text.strip()` is non-empty, prepends `"Context: {context_text}\n\n---\n\n"` to the user message.
       - Calls `_split_into_chunks(text)` to get list of chunks.
       - For each chunk:
         - Sends progress message: `{"type": "claude_chunk", "task_id": task_id, "n": i+1, "total": total_chunks}` for multi-chunk, or `{"type": "claude_processing", "task_id": task_id}` for single-chunk.
         - Starts a daemon thread that sleeps 60s then sends `{"type": "claude_slow", "task_id": task_id, "elapsed": elapsed}` (per D-32).
         - Calls `client.messages.create(model=model, max_tokens=64_000, system=system_prompt, messages=[{"role": "user", "content": chunk_msg}], timeout=timeout)`.
         - Accumulates `response.usage.input_tokens` and `response.usage.output_tokens`.
       - Merges chunks if multiple, returns `{"result": merged, "input_tokens": total_input, "output_tokens": total_output}`.
       - On `anthropic.AuthenticationError`: re-raise with clear message "Invalid API key".
       - On `anthropic.APIStatusError`: re-raise with status code and message.
       - On timeout: re-raise with "Timeout after {timeout}s".

       **`_split_into_chunks(text: str) -> list[str]`:**
       - If `len(text) <= CHUNK_CHARS`, return `[text]`.
       - Otherwise split at `CHUNK_CHARS` boundaries, preferring paragraph breaks (`\n\n`) via `rfind`. Overlap by `OVERLAP_CHARS`.

       **`_merge_chunks(parts: list[str]) -> str`:**
       - Join with `"\n\n---\n\n"`.

       **`validate_api_key(api_key: str) -> tuple[bool, str]`:**
       - Per D-12: send minimal `max_tokens=1` test call to `claude-haiku-4-5`.
       - On `AuthenticationError`: return `(False, "Invalid API key — check the value")`.
       - On `APIStatusError` with status 429 or 529: return `(True, "")` — rate limit still means key is valid.
       - On other `APIStatusError`: return `(False, f"API error {status_code}: {message}")`.
       - On success: return `(True, "")`.

       **`estimate_cost_pre_send(char_count: int, model: str) -> float`:**
       - Per D-29: `estimated_input_tokens = char_count / 4`, `estimated_output_tokens = estimated_input_tokens * 0.8`.
       - Look up pricing from `PRICING` dict, default to haiku pricing.
       - Return `estimated_input_tokens * p["input"] + estimated_output_tokens * p["output"]`.

       **`calculate_actual_cost(input_tokens: int, output_tokens: int, model: str) -> float`:**
       - Per D-30: look up pricing, return exact cost.

       **`generate_diff(original: str, cleaned: str) -> str`:**
       - Use `difflib.unified_diff(original.splitlines(keepends=True), cleaned.splitlines(keepends=True), fromfile="Original", tofile="Cleaned", lineterm="")`.
       - Return joined diff as string. Per D-07: this diff is appended to `_upraveno.txt`.
