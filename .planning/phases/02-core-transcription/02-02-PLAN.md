**Class structure:**

```python
class TranscriptionPanel:
    """File queue, action bar, and log panel for transcription workflow.

    Pure UI — transcription logic wired by Plan 03's dispatcher.
    """

    AUDIO_EXTENSIONS = (".mp3", ".wav", ".m4a", ".ogg")

    def __init__(self, parent: ttk.Frame, root: ttk.Window) -> None:
        self.parent = parent
        self.root = root
        self._row_data: dict[str, dict] = {}  # iid -> {full_path, error_msg, result_text}
        self._output_dir = tk.StringVar(value="")
        self._running = False
        self._build_ui()
```

**Public methods for Plan 03 to call (API contract):**

```python
def append_log(self, message: str, tag: str = "info") -> None:
    """Append a timestamped line to the log panel. Thread-safe via root.after if needed."""

def add_file(self, file_path: Path) -> str | None:
    """Add a file to the queue. Returns iid or None if duplicate/invalid extension."""

def update_row_status(self, iid: str, status: str, tag: str, detail: str = "") -> None:
    """Update the status column and tag for a row."""

def update_row_progress(self, iid: str, n: int, total: int) -> None:
    """Update status cell to show 'zpracovava se... N/T' for the active row."""

def mark_row_done(self, iid: str) -> None:
    """Set row to 'hotovo' status with done tag."""

def mark_row_error(self, iid: str, error_msg: str) -> None:
    """Set row to 'chyba' status with error tag. Store error_msg for tooltip."""

def get_waiting_files(self) -> list[tuple[str, str]]:
    """Return list of (iid, full_path) for all rows with 'waiting' tag."""

def set_transcribing(self, active: bool) -> None:
    """Toggle UI between idle and transcribing states (button label, disable/enable)."""

def update_overall_progress(self, done: int, total: int) -> None:
    """Update the progress bar and 'X / Y souboru' label."""

def get_output_dir(self) -> str:
    """Return current output directory. Empty string means 'same as source file'."""
```

**File picker callbacks (FILE-01, FILE-02):**

`_on_add_files`: Uses `tkinter.filedialog.askopenfilenames` with filetypes filter for AUDIO_EXTENSIONS. For each selected file, calls `self.add_file(Path(f))`.

`_on_add_folder` (FILE-02): Uses `tkinter.filedialog.askdirectory`. Scans directory for files matching AUDIO_EXTENSIONS using `Path.glob("*")`, calls `self.add_file()` for each.

`_on_remove_selected` (FILE-04): Gets `self.tree.selection()`, removes each from tree and `_row_data`. Guards against removing rows with "processing" tag.

`_on_browse_output` (FILE-06): Uses `tkinter.filedialog.askdirectory`, updates `self._output_dir`.

**Save-as context menu (FILE-05, D-10):**

Bind right-click (`<Button-3>` on Windows, `<Button-2>` on macOS) on tree. If clicked row has "done" tag, show context menu with `_("ui.menu.save_as")`. On selection, open `asksaveasfilename` with `.txt` filter. Write `_row_data[iid]["result_text"]` to chosen path.

Also bind `<Double-1>` on tree — if row is "done", trigger the same save-as flow.

**File size formatting:**

Helper function `_format_filesize(size_bytes: int) -> str` returns human-readable string: "1.4 MB", "320 KB", etc. Uses 1024-based units (IEC). Show one decimal place.

**Duration column:**

When a file is added, the `duration` cell initially shows "..." (loading). Use `self.root.after(0, ...)` to schedule an async ffprobe call via `threading.Thread` that runs `_probe_duration(iid, path)`. On completion, updates the cell. If ffprobe fails or binary not found, shows "\u2013" (en-dash). Duration format: "MM:SS".

The ffprobe call pattern (from RESEARCH Pattern 5):
```python
import subprocess, json
from src.whisperai.utils.resource_path import get_resource_path

ffprobe_path = get_resource_path("bin/ffprobe.exe" if sys.platform == "win32" else "bin/ffprobe")
# If ffprobe not found, just show en-dash without error
```

IMPORTANT: Do NOT block the main thread during ffprobe. Run in a background thread.
